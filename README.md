# My wish list

## Course project on the Tinkoff Fintech course, Frontend, 2023
github @Alberthepooh

Link: https://coursework-wishlist.web.app

## Project Description
The application allows you to create lists of gifts for the holidays.

Main functions:
- Adding gifts and sorting them by holidays
- Ability to add a link to a specific gift
- Deleting, editing gifts, marking them as "ready" and "favorites"
- Choosing deadlines for gifts for today
- Personal account and authorization
